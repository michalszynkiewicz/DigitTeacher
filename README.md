## Installation
Clone the repository:
```shell script
git clone https://github.com/michalszynkiewicz/letter-teacher
```

Build the project:
```shell script
./gradlew build
```

Copy the file located in `./app/build/outputs/apk/debug/app-debug.apk` to your phone and install the app from it.
